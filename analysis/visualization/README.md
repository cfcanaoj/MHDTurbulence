
### 2. Full data visualization: XMF + binary (for VisIt / ParaView)
For full 3D visualization, the code outputs binary data together with
XMF metadata files.

Generated files:

    bindata/field?????.xmf
    bindata/field?????.bin
    bindata/grid1D.bin
    bindata/grid2D.bin
    bindata/grid3D.bin

Open the `.xmf` file directly in:

-   VisIt
-   ParaView
