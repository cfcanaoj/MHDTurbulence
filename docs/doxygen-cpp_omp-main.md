# cpp_omp

This site contains the automatically generated API documentation for the
`cpp_omp` implementation of MHDTurbulence.

The source directory `src_cpp_omp/` provides the C++ MPI + OpenMP variant of the
solver. Doxygen builds this site from the C++ headers and source files in that
directory.
